| Клавиша         | Описание                               |
|-----------------|----------------------------------------|
| **Telescope**   |                                        |
| `leader + f`    | Поиск файлов                           |
| `leader + p`    | Поиск проекта                          |
| `leader + b`    | Поиск буфера                           |
| `leader + h`    | Поиск по истории                       |
| `leader + r`    | Поиск текста                           |
| `leader + k`    | Поиск по ключевым картам               |
| **Harpoon**     |                                        |
| `leader + h`    | Меню Harpoon                           |
| `leader + a`    | Добавить файл в Harpoon                |
| `C-1`           | Первый файл Harpoon                    |
| `C-2`           | Второй файл Harpoon                    |
| `C-3`           | Третий файл Harpoon                    |
| `C-4`           | Четвёртый файл Harpoon                 |
| `C-S-P`         | Предыдущий файл Harpoon                |
| `C-S-N`         | Следующий файл Harpoon                 |
| **LSP**         |                                        |
| `gd`            | К определению                         |
| `gr`            | К ссылкам                             |
| `gI`            | К реализации                          |
| `leader + D`    | Тип данных                            |
| `leader + ds`   | Символы документа                     |
| `leader + ws`   | Символы рабочей области               |
| `leader + rn`   | Переименовать                         |
| `leader + ca`   | Действия с кодом                      |
| `gD`            | К объявлению                          |
| `leader + th`   | Вкл./выкл. подсказки                  |
| **Работа с буферами и окнами** |                          |
| `leader + b`    | Открыть буфер                         |
| `leader + x`    | Закрыть буфер                         |
| `leader + X`    | Закрыть все буферы                    |
| `<Tab>`         | Следующий буфер                       |
| `<S-Tab>`       | Предыдущий буфер                      |
| `leader + sv`   | Вертикальное окно                     |
| `leader + sh`   | Горизонтальное окно                   |
| `leader + se`   | Равномерное распределение окон        |
| `leader + sc`   | Закрыть окно                          |
| `<Up/Down>`     | Увеличить/уменьшить окно              |
| `<Left/Right>`  | Увеличить/уменьшить окно              |
| **Прочее**      |                                        |
| `J`             | Соединить строку с следующей          |
| `J/K` (визуальный) | Переместить строки                  |
| `leader + p`    | Вставить без замены                   |
| `leader + d`    | Удалить без копирования               |
| `p` (визуальный)| Вставить, не удаляя                   |
| `leader + g`    | Открыть Neogit                        |
| `leader + rw`   | Заменить слово                        |
| `n/N`           | Следующий/предыдущий поиск + центрировать |
| `<Esc>`         | Снять подсветку поиска                |
| `[d` / `]d`     | Предыдущее / Следующее сообщение диагностики |
| `leader + d`    | Показать сообщение диагностики        |
| `leader + e`    | Открыть Neotree                       |
| `leader + mx`   | Сделать файл исполняемым              |
| `leader + l`    | Lazy plugin manager                   |
| `C-q`           | Выйти из Neovim                       |
